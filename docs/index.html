왜 Comux인가

에이전트 워크플로를 위한 모든 것

여러 에이전트를 동시에 돌리고, 무슨 일이 일어나는지 놓치지 않도록 만들어졌습니다.

워크스페이스 가로 탭

상단 가로 탭으로 워크스페이스를 빠르게 전환. 각 탭의 알림점 색상(실행·완료·중단)으로 상태를 한눈에.

패시브 Claude Code 통합

훅과 CDP 프록시를 자동 설정해 에이전트·도구 활동을 추적. 별도 설정이 필요 없는 제로 컨피그.

내장 브라우저

터미널 옆 패널에서 에이전트의 브라우징을 실시간으로 확인. 스크립트 가능한 API도 제공합니다.

분할 패널

각 탭 안에서 가로·세로로 자유롭게 분할. 탭은 keep-alive로 살아 있어 전환해도 세션이 끊기지 않습니다.

알림 & 상태 표시

에이전트가 주의를 요하면 패널이 빛나고, Windows 토스트와 알림 센터로 전달됩니다.

세션 저장 & 복원

분할 레이아웃·작업 디렉터리·브라우저 URL을 저장하고, 다시 켜면 그대로 복원합니다.

CLI & 파이프 자동화

comux CLI와 named pipe(JSON-RPC)로 워크스페이스·패널·에이전트를 스크립트로 제어합니다.

GPU 가속 & 테마

xterm.js WebGL 렌더링으로 부드러운 출력. Windows Terminal·Ghostty 테마를 그대로 가져올 수 있습니다.

  1. 1압축 해제 — 내려받은 .zip을 원하는 폴더에 풉니다.
  2. 2실행 — 폴더 안의 Comux.exe를 더블클릭합니다.
  3. 3차단 해제 — 처음 차단되면 zip 우클릭 → 속성 → 차단 해제 후 풀어주세요 (Mark of the Web).

CLI 빠른 시작

comux
# 새 워크스페이스 열기
comux new-workspace --title "build"

# 패널을 아래로 분할
comux split --down

# 터미널에 텍스트 전송
comux send "npm run dev"

# 내장 브라우저 열기
comux browser open localhost:5199

# 에이전트 스폰 · 알림
comux agent spawn --label "review"
comux notify "빌드 완료"
